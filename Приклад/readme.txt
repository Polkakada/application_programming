База данных(БД) находится на локальном сервере и хранит информацию о видео. Можно хранить информацию о видео, его авторе, жанре и стране создания видео. Связь с БД осуществляется с помощью модуля psycopg2.
Графические интерфейс написан на PyQT5 и создан с помощью QT Designer. Приложение разбито на файлы по функциям.
Работа с базаой данных осуществляется с помощью creating,del_and_search,inserting,showing файлов.
Графический интерфейс mainWindow,insertDialog,infoDialog,findDialog,delDialog.
В файле main прописана работа с формами и диалогами графического интерфейса, а так же в нем осуществляется сбор информации для дальнейшей обработки в базе данных.
Вывод информации, хранящейся в БД осуществляется в QListWidget главного окна. При нажатии на любой выданный результат открывается окно с полной информацией об этой записи в БД.
